一、使用前环境配置说明

  1. 程序打开方式:
    • windows: 打开md_editor.exe
    • linux: 在终端中cd到程序文件夹下，输入
      ./md_editor
  2. 功能使用扩展依赖:
    • 离线转PDF功能: 需要 wkhtmltopdf离线环境
      安装方式:
      Windows: 官网下载
      Debian系: 
      sudo apt install wkhtmltopdf

      Red Hat/Centos: 
      sudo yum install wkhtmltopdf
